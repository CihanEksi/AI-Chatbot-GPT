# Chat Bot

Developed a chatbot application using Express.js and React, powered by OpenAI. The chatbot generates new questions based on user responses, creating a dynamic and engaging conversation.

# Backend

requirement to run
    - install
        "mongodb-community"

run on terminal
    - npm install
    - npm start
